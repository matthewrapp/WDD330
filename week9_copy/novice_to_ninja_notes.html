  • The Broswer Object Model
    • BOM, A collection of properties and methods that contain info about the browser and comput screen.
    • All the properties and methods are available through the window object.
  • Dialogs
    • window.alert
      • This method will pause the execution of the program and display a message in a dialog box.
    • window.confirm
      • This method will stop the execution of the program and display a confirmation dialog that shows the message provided as an argument, and giving the options of OK or cancel.
      • Returns true if the user clicks ok, and false is the user clicks cancel.
      • Example:
        • window.confirm('Do you wish to continue')
        • Returns:
          a window with 'cancel' and 'ok' is the bottom right
    • window.prompt
      • This method will stop the execution of the program and display a message provided as an argument, as well as an input field that allows the user to enter text.
  • Cookies
    • Small files that are saved locally on a user's computer.
    • Can only be read by a web page from the same domain that set them.
    • Limited to storing up to 4kb of data, 20 cookies per domain
    • Take the form of a text file that contains a list of name/value pairs separated by semicolons.
    • Example:
        • "name=Superman; hero=true; city=Metropolis"
  • Creating Cookies
    • Assign it to JavaScripts "cookie jar" using document.cookie property
    • Example:
      • document.cookie = 'name = Superman';
    • Assigning another cookie to it won't override the entire property, it will just appened to the end of the string. So you add more cookies by assigning them to document.cookie
  • Changing Cookie Values
    • Reassigning it to document.cookie using the same name, but a different value.
  • Cookie Expiry Dates
    • Cookies can be made persistent by adding "; expires=date" to the end of the cookie when set.
    • Date is a date value in the UTC String format
    • An alternative is to set the max-age value (seconds). This takes a value in seconds, but it wasn't supported in IE before v10.
  • The Path and Domain of Cookies
    • By default, cookies can only be read by pages inside the same directory and comain as the file was set.
    • The path can be changed so that any page in the root directory can read the cookie.
    • It's done by adding the string "; path=/" to the end of the cookie when it is set
    • Also, it's possible to set the doamin by adding "; domain = domainName" to the end of the cookie
  • Timing Functions
    • setTimeout()
      • The window object provides used methods for scheduling the execution of a function, and for repeatedly executing functions at regular intervals.
      • The window.setTimeout() method accepts a callback as first paramenter and a number of milliseconds as a second parameter.
    • setInterval()
      • This method works in a similar way, except that it will repeatedly invoke the callback function after every given number of milliseconds.
    • Both of these methods can be used to animate elements on a web page.
