Chapter 8 Notes

  • Transforms
    • lets you translate, rotate, scale, and/or skew any element on the page
    • We can manipulate an element's appearance using transform functions.
  • Translation
    • Allows you to move elements left, right, up or down.
    • Example:
      • transform: translate(45px, -45px)
      • this function moves an element x from the left and y from the top
  • Skew
    • The Skew function specifics a skew along the x & y axis.
    • Example:
      • transform: skew(15 deg, 4 deg);
  • Transitions
    • Transitions allow the values of CSS properties to change over time, providing simple animations.
    • Here are the steps to create a simple transition using only CSS:
      1. Declare the original state of the element in the defualt declation.
      2. Declare the final state of your transitioned element
        • :hover (state), etc.
      3. In the transition functions in your default style declaration using the transition properties, including:
        • transition-property
        • transition duration
        • transition-timing-function
        • transition-delay
    • Important to include a pre-state and a post-state
  • Keyframes
    • To animate an element in CSS, you first create a named animation, then attach it to an element in that element's property declaration block.
    • To create am animation, use the keyframs rule
    • Example:
      • @-webkit-keyframes myAnimation {
        put animation keyframes here }
        @keyframes myAnimation {
        put animation keyframes here
        }
    • Here are a few simple animations:

      • @keyframes moveRight {
        from {
        transform: translateX(-50%);
        }
        to {
        transform: translateX(50%);
        }
        }

        @keyframes appearDisappear {
        0%, 100% {
        opacity: 0;
        }
        20%, 80% {
        opacity: 1;
        }
        }

        @keyframes bgMove {
        100% {
        background-position: 120% 0;
        }
        }
      • The second animation is worth paying extra attention to: we’ve applied the same styles to 0% and 100%, and to 20% and 80%. In this case, it means the element will start out invisible ( opacity: 0; ), fade in to visible by 20% of the way through the duration, remain visible until 80%, then fade out.
